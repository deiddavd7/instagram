PARTOLE CHIAVE
REPOSITORY è una struttura dati un modo per salvare dati
    formata da un insieme di FILE e DIRECTORY
    una REPOSITORY è formata da uno o più BRENCH
    un BRENCH è una linea di sviluppo nel tempo 
    un BRENCH è formato da uno o più COMMIT
    COMMIT è un istantanea del progetto 


PROCEDURA PER NUOVO PROGETTO 

mi sposto all'interno della cartella del nuovo progetto
CREO UNA REPOSITORY git init SI FA ALL'INIZIO DI OGNI NUOVO progetto

in automatico si crea un BRANCH che viene chiamto MASTER
